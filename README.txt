Casa Corazon Bloom — Downloadable Website

What's included
- index.html
- styles.css
- app.js
- assets/garden-bg.svg
- audio/ (empty folder for your mp3 files)

How to run (no build tools)
1) Unzip the folder.
2) Open index.html in your browser.

Optional audio
- Add mp3 files into /audio with these names:
  - kindness-01.mp3
  - kindness-02.mp3
  - kindness-03.mp3
- Or edit the tracks array in app.js.

Customizing
- Update contact email in index.html and app.js (mailto link).
- Replace placeholder downloads/logos with real links/assets.
